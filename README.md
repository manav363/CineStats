# CineStats 🎬
A movie discovery dashboard built with vanilla JavaScript and the TMDB API.

## 🔍 About
CineStats lets users browse trending movies, search by title, filter by genre 
or rating, and save favourites — all in a responsive, clean dashboard UI.

## 🌐 API Used
[The Movie Database (TMDB) API](https://www.themoviedb.org/documentation/api)

## ✨ Planned Features
- 🔎 Search movies by title
- 🎛️ Filter by genre, year, and rating
- 📊 Sort by rating or release date
- ❤️ Add to Favourites (saved in localStorage)
- 🌙 Dark / Light mode toggle
- ⏳ Loading indicators during API calls
- 📄 Pagination for results

## 🛠️ Tech Stack
- HTML5
- CSS3 (Vanilla)
- JavaScript (ES6+)
- TMDB API

## 🚀 How to Run
1. Clone the repo:
   git clone https://github.com/YOUR_USERNAME/cinestats.git
2. Get a free TMDB API key at https://www.themoviedb.org/settings/api
3. Add your API key in js/api.js
4. Open index.html in your browser — no build step needed!
